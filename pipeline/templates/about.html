About this tracker
Who's behind it

This tracker is built and run by Kuba Modrzejewski. It started as a way to answer a simple question mechanically instead of impressionistically: when Germany, France and Poland each publish a foreign-ministry statement in the same week, are they actually saying the same thing? Everything on this site — the ingestion pipeline, the enrichment prompts, the scoring — is one person's project, run daily, with the full history committed to git.

Why it exists

Weimar Triangle coordination between the three capitals is rarely captured by a single joint statement — it shows up as three separate press releases in the same window, and whether they converge or diverge is easy to miss if you're not reading all three every day. This tracker reads them so you don't have to, and puts the positions side by side instead of leaving the comparison to memory.

Methodology, in one paragraph

Every tracked statement is rated by an LLM on a five-point scale (−2 to +2) against the officially agreed Weimar Triangle goals, and every rating is stored with the verbatim quote that justifies it — the same discipline political scientists use to hand-code party manifestos, applied at daily cadence. There is no black-box similarity score behind the headline numbers: you can read the exact sentence a rating is based on. All source data lives as YAML files committed to git, so the full audit trail — what was published, when it was ingested, how it was scored — is diffable history, not a database you have to trust. See the full methodology for the scoring rubric and source list.

Corrections policy

Stance ratings are model-assigned and always quote-backed, but models make mistakes. If a rating looks wrong, if a quote is out of context, or if a statement was misclassified, please report it — corrections are reviewed and the underlying YAML is corrected in git, so the fix is itself part of the public record.
